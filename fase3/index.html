Propósito del Sitio


En este sitio se encuentra una descripción y el manejo del Flex Box y CSS Grid, encontrará conceptos de cada uno de ellos, su funcionalidad, para qué están diseñados y cuál es su utilidad. Se podrá observar lo interasante de estas funciones que se podrán utilizar para el incluirlas en el diseño de un sitio web y que tengas más opciones en el momento de la creación y aplicación, aprovecharemos cada una de las herramientas dadas por estas funciones.


Flexbox


Conceptos Básicos de felxbox


El módilo de Caja Flexible que es llamado flexbox es un diseño unidimencional de layout, también es utilizado como método que distribuye el espacio entre los ítems de una interfaz y la capacidad de alineación. Cuando se habla de unidimensional, de indica que el flexbox maneja el layout en una sola dimensión a la vez, ya sea como una fila o una columna. Esto hace un contraste con el modelo bidimencional Grid Layout de CSS el cual realiza un control entre filas y columnas.
Conceptos Básicos de flexbox. MDN Web Docs. Recuperado de Referencia


Actualmente felxbox es compatible con los navegadores más importantes y diferentes versiones de los mismos. En algunos es necesario utilizar prefijos css para que se genere un mejor funcionamiento. Si se llega a presentar algún pronblema con las versiones anteriores de sus navegadores se puede probar el Autoprefixer.
Hasta el momento son conocidos cuatro modos para disponer los elementos:


– Elementos en línea (display:inline)
– Elementos en bloque (display:block)
– Elementos en tabla (display:table)
– Tipos de posición (position:absolute/relative/fixed)

El W3C define a esta estructura como un “flujo flexible” de elementos en dirección (arriba/abajo/izquierda/derecha) y tamaño (anchura/altura) según los ejes principal (horizontal) y transversal (vertical).

CSS Grid Layout


Tiene funciones de diseño dirigidas para el desarrollo de aplicaciones web. Éste puede ser utilizado multiples diseños. También es destacado porque permite dividir una página en áreas o regiones principales, define la relación en términos de tamaño, posición y capas entre las partes de un cotrol que es contruido a partir de hatml nativo. El Grid Loyout también permite alinear elementos en columnas y filas, pero no solo para ello es utilizado el Grid Loyout, es posible realizar muchos más diseños y de una forma más sencilla que con las tablas.
CSS Grid Layout. MDN Wev Docs. Recuperado de Referencia
Cuando hablamos de crear un sitio web, la mayor parte de del trabajo consiste en disponer distintos elementos que lo componen. El diseño debe ser algo que atraiga. En CSS las hojas de estilo en cascada constituye una herramienta para diseñar sitios web con ciertas características, mientras que en HTML muestra un contenido de manera rudimentaria y css es apto para diseños más complejos.
En el mundo del internet móvil encontramos un inumerable de desafíos para los diseñadores web, debido a esta variedad de diseños es muy complejo saber cuál es el formato que tiene la pantalla en la que se va a visualizar el contenido web. Por esto es tan importante los elementos individuales como cajas de texto, gráficos, elementos interactivos, sean destribuidos de manera independiente y de forma muy clara, teniento muy presente las respectivas condiciones de espacio dadas por la pantalla. A diferencia del Felx box, que como mencionamos, es unidimensional, CSS Grid ofrece dos diemnsiones para la colocación de los objetos, porque en lugar de tener solo un eje, permite la creación de rejillas con filas y columnas.
